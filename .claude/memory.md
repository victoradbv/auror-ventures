# Project Memory

## What this is
Astro 6 + Tailwind CSS 4 corporate website for **Auror Ventures** (software consulting).
Bilingual: English (`src/pages/en/`) and Spanish (`src/pages/es/`) — always mirror changes to both.

## Stack
- Astro 6, Tailwind CSS v4 (no config file — tokens in `src/styles/global.css`)
- Local fonts: Uni Neue (headings), Roboto (body/UI)
- Static only — no backend, no CMS, no image optimization pipeline

## Design tokens (global.css)
- Primary: `#10566f`, Accent: `#207E82`, Container: `1100px`
- Shared classes: `.btn-primary`, `.container-custom`, `.font-heading`, `.font-ui`, `.font-body`
