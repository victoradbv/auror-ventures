# Auror Ventures Website

Official website for Auror Ventures, built with Astro and Tailwind CSS.

## Tech Stack

* Astro
* Tailwind CSS

## Development

npm install
npm run dev

## Build

npm run build

## Structure

* `src/components` → UI components (Header, Footer, etc.)
* `src/layouts` → Page layouts
* `src/pages` → Routes
* `public` → Static assets (images, fonts)

## Notes

* Fonts are locally hosted
* Styling follows a Tailwind-first approach
* Design system uses CSS variables for colors
